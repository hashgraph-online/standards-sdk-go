# HCS-16: Build a Flora Topic Transaction

This example builds an [HCS-16](https://hol.org/docs/standards/hcs-16) flora topic create transaction.

## What it does

1. Constructs an HCS-16 flora topic with threshold-key multi-party consensus.
2. Demonstrates the collaborative governance setup.

## Run

```bash
export HEDERA_ACCOUNT_ID="0.0.xxxxx"
export HEDERA_PRIVATE_KEY="302..."
go run ./examples/hcs16-build-flora-topic-tx
```
