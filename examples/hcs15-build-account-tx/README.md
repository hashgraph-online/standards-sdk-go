# HCS-15: Build a Base Account Transaction

This example builds an [HCS-15](https://hol.org/docs/standards/hcs-15) base account create transaction.

## What it does

1. Constructs an HCS-15 base account creation transaction.
2. Demonstrates hierarchical account setup for petal accounts.

## Run

```bash
export HEDERA_ACCOUNT_ID="0.0.xxxxx"
export HEDERA_PRIVATE_KEY="302..."
go run ./examples/hcs15-build-account-tx
```
