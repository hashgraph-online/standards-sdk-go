# HCS-7: Register Metadata

This example registers [HCS-7](https://hol.org/docs/standards/hcs-7) metadata in an indexed registry.

## What it does

1. Registers metadata entries in an HCS-7 indexed registry topic.
2. Demonstrates the metadata registration workflow.

## Run

```bash
export HEDERA_ACCOUNT_ID="0.0.xxxxx"
export HEDERA_PRIVATE_KEY="302..."
go run ./examples/hcs7-register-metadata
```
