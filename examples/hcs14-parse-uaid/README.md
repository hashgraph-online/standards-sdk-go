# HCS-14: Parse a Universal Agent ID

This example parses and canonicalizes an [HCS-14 Universal Agent ID (UAID)](https://hol.org/docs/standards/hcs-14).

## What it does

1. Parses a UAID string into its structured components.
2. Rebuilds the canonical UAID representation.
3. Prints the target, ID, and canonical form.

## Run

```bash
go run ./examples/hcs14-parse-uaid
```
