# HCS-21: Build an Adapter Declaration

This example builds an [HCS-21](https://hol.org/docs/standards/hcs-21) adapter declaration transaction.

## What it does

1. Constructs an HCS-21 adapter declaration for the Registry Broker.
2. Demonstrates how to declare adapter capabilities on-chain.

## Run

```bash
export HEDERA_ACCOUNT_ID="0.0.xxxxx"
export HEDERA_PRIVATE_KEY="302..."
go run ./examples/hcs21-build-declaration
```
