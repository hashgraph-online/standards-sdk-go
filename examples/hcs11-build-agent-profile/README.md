# HCS-11: Build an AI Agent Profile

This example builds and serializes an [HCS-11](https://hol.org/docs/standards/hcs-11) AI agent profile.

## What it does

1. Uses the AgentBuilder to construct a typed AI agent profile.
2. Serializes the profile to JSON for inscription on the Hedera ledger.

## Run

```bash
go run ./examples/hcs11-build-agent-profile
```
