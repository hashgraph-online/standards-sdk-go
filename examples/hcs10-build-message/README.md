# HCS-10: Build a Connection Message

This example builds an [HCS-10](https://hol.org/docs/standards/hcs-10) connection message transaction.

## What it does

1. Constructs an HCS-10 connection message for agent communication.
2. Demonstrates the OpenConvAI messaging format.

## Run

```bash
export HEDERA_ACCOUNT_ID="0.0.xxxxx"
export HEDERA_PRIVATE_KEY="302..."
go run ./examples/hcs10-build-message
```
