# HCS-18: Build an Announce Discovery Transaction

This example builds an [HCS-18](https://hol.org/docs/standards/hcs-18) announce discovery transaction.

## What it does

1. Constructs an HCS-18 announce message for agent/service discovery.
2. Demonstrates the discovery announcement workflow.

## Run

```bash
export HEDERA_ACCOUNT_ID="0.0.xxxxx"
export HEDERA_PRIVATE_KEY="302..."
go run ./examples/hcs18-build-announce
```
