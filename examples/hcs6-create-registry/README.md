# HCS-6: Create a Dynamic Hashinal Registry

This example creates a non-indexed [HCS-6](https://hol.org/docs/standards/hcs-6) dynamic hashinal registry.

## What it does

1. Creates an HCS-6 dynamic hashinal registry topic on testnet.
2. Demonstrates the non-indexed registry creation workflow.

## Run

```bash
export HEDERA_ACCOUNT_ID="0.0.xxxxx"
export HEDERA_PRIVATE_KEY="302..."
go run ./examples/hcs6-create-registry
```
