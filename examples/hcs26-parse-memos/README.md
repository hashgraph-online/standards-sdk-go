# HCS-26: Parse Topic and Transaction Memos

This example builds and parses [HCS-26](https://hol.org/docs/standards/hcs-26) topic and transaction memos.

## What it does

1. Constructs structured memos following the HCS-26 format.
2. Parses existing memos back into structured components.

## Run

```bash
go run ./examples/hcs26-parse-memos
```
