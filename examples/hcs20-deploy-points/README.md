# HCS-20: Deploy a Private Points Topic

This example deploys a private [HCS-20](https://hol.org/docs/standards/hcs-20) points topic.

## What it does

1. Creates a private HCS-20 points topic on testnet.
2. Demonstrates the points system deployment workflow.

## Run

```bash
export HEDERA_ACCOUNT_ID="0.0.xxxxx"
export HEDERA_PRIVATE_KEY="302..."
go run ./examples/hcs20-deploy-points
```
