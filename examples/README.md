# Examples

This folder contains runnable examples for common SDK flows.

- `hcs2-create-registry`: Creates an indexed HCS-2 registry topic.
- `hcs5-build-mint`: Builds an HCS-5 mint transaction with HCS-1 HRL metadata.
- `hcs11-build-agent-profile`: Builds and serializes an HCS-11 AI agent profile.
- `hcs14-parse-uaid`: Parses and canonicalizes a UAID string.
- `hcs15-build-account-tx`: Builds an HCS-15 base account create transaction.
- `hcs16-build-flora-topic-tx`: Builds an HCS-16 flora topic create transaction.
- `hcs17-build-state-message`: Builds an HCS-17 state hash message transaction.
- `hcs20-deploy-points`: Deploys a private HCS-20 points topic.
- `hcs27-publish-checkpoint`: Publishes an HCS-27 checkpoint message.
- `inscriber-auth-client`: Authenticates against Kiloscribe and initializes an inscriber client.

Run any example from repo root:

```bash
go run ./examples/hcs2-create-registry
```
