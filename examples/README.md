# Examples

Runnable examples for common [HOL Standards SDK](https://hol.org/docs/libraries/standards-sdk/) flows. Each example targets a specific [Hiero Consensus Specification](https://hol.org/docs/standards).

| Example | Specification | Description |
| ------- | ------------- | ----------- |
| `hcs2-create-registry` | [HCS-2](https://hol.org/docs/standards/hcs-2) | Creates an indexed topic registry |
| `hcs5-build-mint` | [HCS-5](https://hol.org/docs/standards/hcs-5) | Builds a Hashinal mint transaction |
| `hcs6-create-registry` | [HCS-6](https://hol.org/docs/standards/hcs-6) | Creates a dynamic hashinal registry |
| `hcs7-register-metadata` | [HCS-7](https://hol.org/docs/standards/hcs-7) | Registers metadata in an indexed registry |
| `hcs10-build-message` | [HCS-10](https://hol.org/docs/standards/hcs-10) | Builds a connection message |
| `hcs11-build-agent-profile` | [HCS-11](https://hol.org/docs/standards/hcs-11) | Builds an AI agent profile |
| `hcs12-build-register` | [HCS-12](https://hol.org/docs/standards/hcs-12) | Builds a register payload |
| `hcs14-parse-uaid` | [HCS-14](https://hol.org/docs/standards/hcs-14) | Parses a Universal Agent ID |
| `hcs15-build-account-tx` | [HCS-15](https://hol.org/docs/standards/hcs-15) | Builds a base account transaction |
| `hcs16-build-flora-topic-tx` | [HCS-16](https://hol.org/docs/standards/hcs-16) | Builds a flora topic transaction |
| `hcs17-build-state-message` | [HCS-17](https://hol.org/docs/standards/hcs-17) | Builds a state hash message |
| `hcs18-build-announce` | [HCS-18](https://hol.org/docs/standards/hcs-18) | Builds an announce discovery message |
| `hcs20-deploy-points` | [HCS-20](https://hol.org/docs/standards/hcs-20) | Deploys a private points topic |
| `hcs21-build-declaration` | [HCS-21](https://hol.org/docs/standards/hcs-21) | Builds an adapter declaration |
| `hcs26-parse-memos` | [HCS-26](https://hol.org/docs/standards/hcs-26) | Parses topic/transaction memos |
| `hcs27-publish-checkpoint` | [HCS-27](https://hol.org/docs/standards/hcs-27) | Publishes a Merkle checkpoint |
| `inscriber-auth-client` | [HCS-1](https://hol.org/docs/standards/hcs-1) | Authenticates with Kiloscribe |

## Run

```bash
go run ./examples/hcs2-create-registry
```
